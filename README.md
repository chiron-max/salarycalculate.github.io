# 月度工作时长统计网站使用手册

https://chiron-max.github.io/Salary-Calculate/

本系统是一款专业的月度工作时长统计工具，专为需要精确记录工作时间的用户设计。支持夜班、跨天、加班、碎片化工时记录，以及多日期批量操作，帮助您轻松管理每月工作时间。

## 快速上手

### 1. 记录工作时间
- **单日记录**：点击月历中的日期，填写开始/结束时间，选择工种和类型（正常/加班/夜班）
- **批量操作**：
  - 点击"多选日期"，按住`Ctrl`/`Shift`选择多个日期（未实现）
  - 为选中日期批量添加相同时间段
  - 支持"本周"、"工作日"等快捷选择

### 2. 工种配置（首次使用必看）
- 点击"⚙ 工种管理"
- **新建工种**：设置名称、默认时间、预期工时、夜班窗口
- **夜班设置**：配置"夜班窗口开始/结束"时间，系统自动计算夜班时长

### 3. 数据管理
- **修改/删除**：点击日期进入编辑，或批量选择后删除
- **数据导出**：支持`JSON`（技术人员）和`CSV`（Excel兼容）格式
